# Chart-Maker

Using the chart maker we can create charts of any of the following types :
1. Bar Chart
2. Stacked Bar Chart
3. Pie Chart
4. Line Chart
5. Donut Chart
This project provides an option of printing the charts and is implemented using Qt5 GUI Library in C++.
